# Rule-Based Therapist Bot

A simple chatbot that detects emotional keywords and responds with
empathetic statements using rule-based classification.

---

## Features
- Detects emotional words (sad, angry, happy, stressed)
- Cleans and processes user input
- Continuous conversation until exit
- Default fallback responses

---

## Concepts Used
- Dictionaries
- Nested loops
- String normalization
- Conditional logic

---

## Example

How are you feeling? I feel stressed about work
That sounds overwhelming. Want to talk about what's causing it?

---

## Run
python main.py
