# Number Guess Game

A classic guessing game where the player attempts to guess a randomly
generated number within a limited number of attempts.

---

## Features
- Random number generation
- High/Low hints
- Attempt counter
- Replay option

---

## Concepts Used
- Random module
- Loops
- Conditionals
- Integer conversion

---

## Example

Guess a number between 1 and 100: 50
Too high!
Guess again: 25
Too low!
Guess again: 37
Correct! You guessed in 3 tries.

---

## Run
python main.py
