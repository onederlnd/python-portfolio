# Adventure Story Game

An interactive text adventure where player choices determine
the story outcome.

---

## Features
- Branching story paths
- Multiple endings
- Choice validation
- Replay option

---

## Concepts Used
- Nested conditionals
- Game state tracking
- User input handling

---

## Example

You enter a dark forest.
1) Go left
2) Go right

Choice: 2
You encounter a river...

---

## Run
python main.py
