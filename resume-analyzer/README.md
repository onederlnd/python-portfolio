# Resume Analyzer

A text-processing program that reads a resume file and analyzes
skills, experience keywords, and word frequency.

---

## Features
- Reads resume from text file
- Counts keywords
- Displays detected skills
- Basic scoring system

---

## Concepts Used
- File reading
- String parsing
- Dictionaries and counting
- Data reporting

---

## Example

Skills Found:
Python: 4
SQL: 2
Leadership: 1

Resume Strength Score: 72/100

---

## Run
python main.py
