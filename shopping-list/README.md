# Shopping List with Categories

A command-line shopping list manager that allows users to organize grocery items
by category and save/load them from a file.

---

## Features
- Add items to categories
- Remove items
- View organized shopping list
- Save list to file
- Load list from file

---

## Concepts Used
- Dictionaries
- Lists
- File read/write
- Loops and menus
- Input validation

---

## Example

1. Add Item
2. Remove Item
3. View List
4. Save
5. Load
6. Exit

Choose: 1
Enter category: Dairy
Enter item: Milk
Item added.

---

python main.py
